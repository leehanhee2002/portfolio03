2D 격투 게임 개발 프로젝트

원작 게임

개인 제작

개발 기간 2021 ~ NOW
개발 인원 1인
사용 기술 C 언어
개발 목적 국내에 출시되지 않은 3D 격투 게임을 2D 스타일로 구현하기.
작품 설명 해당 프로젝트는 국내에 출시되지 않은 3D 격투 게임을 2D 스타일로 재구현하는 게임 개발 프로젝트입니다. 원작 게임의 스토리 모드 플레이 영상을 시청해 캐릭터의 동작 구조와 전투 시스템을 분석한 뒤, 2D 환경에서도 동일한 타격감과 조작성을 구현하기 위해 애니메이션 분석, 프레임 단위의 타이밍 조정, 콜라이더 세분화 등 여러 기술적 시도를 반복했습니다. 이러한 과정을 통해 원작의 핵심 재미 요소를 유지한 2D 버전을 성공적으로 완성했으며, 관련 시연 영상을 개인 유튜브 채널에 꾸준히 업로드하며 개선을 이어가고 있습니다. 또한 원작에 등장하지 않은 캐릭터와 기술을 오픈소스 리소스를 활용해 제작하고 게임에 추가함으로써, 기존에는 볼 수 없던 새로운 매치업을 구현했습니다. 이를 통해 프로젝트 공개 후 다양한 조합을 즐길 수 있다는 긍정적인 반응을 얻었습니다.

정보 프로그램 제작 프로젝트

개발 연도 2022년
개발 인원 1인
사용 기술 Unity(C#), Scratch
작품 설명 검색창에 2010년에서 2022년 사이의 연도를 입력한 다음, 검색하면 해당 연도에 개봉한 10가지 영화의 줄거리와 감독, 출연진 정보를 제공해주는 프로그램입니다. Unity 버전과 Scratch 버전 총 2개를 만들었습니다. 시연 과정에서 체험자가 2021년을 입력했을 때, 프로그램이 제시한 영화들이 체험자의 취향과 맞아 긍정적인 평가를 받은 경험이 있어 데이터 활용과 사용자 경험의 중요성을 다시 한 번 느낄 수 있었습니다.

점프 액션 탈출 게임 제작 프로젝트

이 프로젝트는 컴퓨터활용능력 2급 필기 시험을 주제로 점프 액션 장르를 결합한 게임 개발 프로젝트입니다. 개발 과정에서 (주) 드림아이디어소프트의 문창규 멘토 님과 3차례의 멘토링 시간을 가졌고 이후 완성한 게임의 자세한 사항은 아래와 같으며 게임 공모전에도 참가해 결선까지 진출했습니다.

프로젝트 정답을 향해 점프 (영문명 : Quiz and Jump)
장르 교육, 탈출, 점프 액션
제작 팀 트리오
기술 Unity (C#)
개발 기간 2024.03 ~ 2024.06
게임 설명 배경에 제시되는 문제를 풀고 방향키로 고양이 캐릭터를 조작해 정답 번호가 적힌 깃발을 향해 뛰어가 탈출하는 게임으로 총 20개의 스테이지로 구성되어 있습니다. 문제는 실제 시험과 동일하게 전부 객관식이고 보기도 4개입니다. 화면 구성은 총 4개로 시작 화면, 게임 설명 화면, 스테이지 선택 화면, 20개의 스테이지 화면입니다. 실수로 고양이가 구름 아래로 추락해도 다시 시작 지점으로 되돌아갑니다.
